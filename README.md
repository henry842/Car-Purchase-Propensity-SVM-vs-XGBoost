# Car Purchase Propensity – SVM vs XGBoost

Projeto de Machine Learning para análise de propensão à compra de carros.

## Modelos Utilizados
- XGBoost
- SVM com kernel Linear
- SVM com kernel Polinomial

## Etapas
- Análise exploratória
- Tratamento de dados
- Treinamento de modelos
- Avaliação com métricas de classificação
- Comparação de desempenho entre algoritmos

## Tecnologias
- Python
- Pandas
- Scikit-learn
- XGBoost
- Matplotlib / Seaborn
